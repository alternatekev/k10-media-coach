# iRacing Progression Tracking — Feature Brainstorm

Goals: 4000 iRating · A Sportscar License · A Formula License

## Overlay (HUD during sessions)

### iRating Tracker
Show current iRating with a small spark-line of recent trend. After each race, flash the delta (+/-) so you see the gain or loss immediately. A subtle progress bar toward 4000 could sit in a corner.

### Safety Rating Live Monitor
Real-time incident count for the current session with a projected SR change. Color-code it: green when you're on track for a gain, amber when you're close to your budget, red when you've blown it. Since license promotion depends on SR thresholds, showing "incidents remaining before SR drops below X" is more actionable than just the raw count.

### License Promotion Checklist
A compact widget showing what's left for each license class (minimum races, MPR, SR threshold). Especially useful for the sportscar and formula paths since they're separate progressions you're running in parallel.

### Session Goal Reminder
Before going green, flash a short goal you set on the web ("clean first 5 laps", "no 4x this race", "top-split pace by June"). Keeps the mindset intentional.

### Pace Delta Overlay
Not just the standard race delta, but one that compares your rolling pace against the pace profile you'd need to be competitive at the 4000 iR split level. "You're 0.6s off 4k pace" is a powerful motivator.

## Web Dashboard

### Progression Timeline
A chart with three lanes (iRating, sportscar license, formula license) plotted over time. Mark milestones, setbacks, and the projected date to each goal based on current trajectory.

### Race Journal
After each session, prompt for a quick note (what went well, what to work on). Auto-populate with the telemetry stats so the entry is lightweight. Over time this becomes a searchable log of lessons learned.

### Series Planner
Map out which series feed each license path, which ones are in-season, and which offer the best iR gain opportunity for your current skill level. Participation counts and strength-of-field data could inform recommendations like "Road America Sportscars has weak SoF on Tuesday mornings."

### Incident Analysis
Pull incident data per race and categorize it (contact, off-track, loss of control). Show patterns: are you collecting 4x's at starts? In traffic? At specific tracks? That's the fastest lever for SR improvement.

### iRating Simulator
"What if" tool: if you finish P5 in a 2800 SoF race, how much iR do you gain? Helps with series selection and race-by-race strategy toward 4000.

### Track Preparation Checklist
For each upcoming week's track, surface your historical data there (best lap, average incidents, finish positions) and link to setup resources. Knowing your weak tracks in advance is huge.

### Dual-Path License Roadmap
Since sportscar and formula are separate grinds, a visual roadmap showing where you are on each, what class you need to run next, and whether you should prioritize one path over the other this week based on schedule and SR headroom.

### Milestone Sharing
When you hit a goal (B license, 3000 iR, first top-split race), generate a shareable card with your stats for socials or Discord.
